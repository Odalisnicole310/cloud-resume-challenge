# Cloud Resume Challenge using Azure

This project is a cloud-hosted resume website built and deployed using Microsoft Azure and GitHub.

## Live Demo

Azure Static Website: https://odaliscloud10.z20.web.core.windows.net/

## Technologies Used

* Microsoft Azure Storage Static Website
* GitHub
* GitHub Actions
* GitHub Secrets
* HTML
* CI/CD

## What I Built

* Created a static resume website using HTML.
* Hosted the website using Azure Storage Static Website.
* Configured GitHub Actions to automatically deploy updates from GitHub to Azure.
* Stored Azure deployment credentials securely using GitHub Secrets.
* Verified the deployment through a successful GitHub Actions workflow.
